ServerManager Framework 🚀

A custom-built, Express-like HTTP server framework created entirely from scratch using Node.js native net module.

Core Infrastructure

This server handles raw TCP connections and HTTP protocols manually, providing a clean API for developers:

  • Routing Engine: Clean separation of GET and POST requests.
  • Request Parser: Automatically parses query parameters and extracts JSON payloads into req.body.
  • Smart Responses: Chainable response methods (res.status(200).json(...)).
  • Static Files: Built-in serveStaticFile method with MIME type detection and 404 error handling.

✨ Creative Features (Bonus)

Beyond the standard requirements, this framework includes two unique implementations:

1. Real-Time File Logger

A built-in middleware-style logger that intercepts incoming requests. It extracts the timestamp, HTTP method, requested path, client IP address, and the parsed JSON body, writing them asynchronously to a local server.log file.

2. Interactive API Game (HTTP Wordle)

A fully playable, server-validated Wordle clone. The client sends guesses via POST requests, and the custom API validates the input against the secret word, computing the hit/miss array dynamically on the backend.
